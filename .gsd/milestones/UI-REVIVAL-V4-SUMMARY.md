## Lessons Learned
- Tailwind v4 `@theme` blocks are highly efficient for managing multi-theme CSS variables.
- Pure CSS transitions on `body` and `header` provide a premium feel without complex animation libraries.
- Mobile drawer logic requires careful handling of fixed positioning and z-index to ensure visibility of over-the-sidebar elements (like tooltips) when expanded.
