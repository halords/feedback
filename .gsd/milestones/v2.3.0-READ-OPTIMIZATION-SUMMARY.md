## Lessons Learned
- **High-Density UI Power**: Compact layouts significantly reduce visual fatigue for administrative users when handling large datasets.
- **Context-Aware Fetching**: Passing month/year context to utility APIs (like `offices`) is more robust than global client-side filtering.
- **Graceful Desync Handling**: Always synchronize sub-filters when a parent filter (like Year) changes to prevent ghost state selections.
