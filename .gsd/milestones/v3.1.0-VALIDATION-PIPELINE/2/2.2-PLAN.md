# Plan 2.2: High-Performance Service Refactor

## Objective
Refactor the service layer to abandon expensive full-collection scans in favor of highly efficient, indexed Firestore queries using the newly migrated ISO date fields.

## Context
- src/lib/services/responseService.ts
- src/lib/services/metricsService.ts
- src/lib/services/physicalReportService.ts
