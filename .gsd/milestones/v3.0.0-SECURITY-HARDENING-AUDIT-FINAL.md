## Concerns
- **Automated Testing Gap**: All security verification is currently manual. Lack of automated RBAC unit/integration tests poses a regression risk during future feature development.
- **Legacy UI Density**: The high-density "Industrial Slate" theme requires an accessibility audit for WCAG contrast compliance.

## Recommendations
1. **Infrastructure Verification**: Confirm Firestore Security Rules are synced with code-level RBAC for defense-in-depth at the data layer.
2. **Automated RBAC Suite**: Develop a suite of tests that mock `user_type` and `offices` to ensure scoping logic in `rbac.ts` remains intact.
3. **Zod Migration**: Shift `apiSchemas.ts` to Zod to improve error reporting and type safety.
