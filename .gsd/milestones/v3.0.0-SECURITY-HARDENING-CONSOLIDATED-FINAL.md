## Concerns
- **Infrastructure Sync**: Security depends on both code-level RBAC and Firestore Rules. It is recommended to verify Firestore Rules in a separate audit to ensure defense-in-depth is 100% synced.

## Recommendations
1. **Automated Regression Suite**: Prioritize adding unit tests for `rbac.ts` in the next milestone to ensure these security gains never regress.
